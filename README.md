## 📘 Know How Note 📘

## 📝 概要
 仕事中やプライベートで得たノウハウを一括管理し、共有できるWebサイトです

## 📚 アプリ機能
 **1.ノウハウの作成**
 　　- ノウハウを作成し、フォルダごとにまとめて一括管理
    
 **2.ノウハウの共有**
　　 - 個々人で管理しているノウハウを任意のユーザーに共有可能 

## 💻 開発環境

- HTML  
- CSS  
- Javascriprt
- Node.js  
- MySQL 
- HEROKU
- VSCode（Visual Studio Code）
